축구 콘텐츠 제작소 ‘score90’은 베팅 사이트 Bet365의 정보를 빌려 맨체스터 시티의 유럽축구연맹(UEFA) 챔피언스리그(UCL) 우승 확률을 공개했다. 맨시티의 UCL 우승 확률은 78%다. 잉글리시 프리미어리그(EPL), FA컵까지 모두 정복하는 트레블 달성 가능성이기도 하다. 맨시티는 이번 시즌 엄청난 퍼포먼스를 선보이는 중이다. 아스널과의 경쟁에서 밀리는 양상을 보이면서 EPL 우승을 내주는 분위기가 컸다. 하지만, 아스널이 삐걱거리는 틈을 이용해 결과를 챙기면서 맹추격에 나섰고 역전 우승의 짜릿함을 맛봤다.
